## About The Project

I have created a fixed bottom navigation bar with Tailwindcss and HTML.

Here's why:
* I was searching for something like this for hours, but I haven't found anything. So I made it.
* You shouldn't have to make a fixed bottom navigation bar, because you should focus on important things.

### Built With

* [Tailwindcss](https://tailwindcss.com)
